¿Qué es SEO (Search Engine Optimization)?

SEO es la sigla para Search Engine Optimization, que significa "optimización para motores de búsqueda". Consiste en una serie de técnicas, disciplinas y estrategias de optimización que se implementan en las páginas de un sitio web o blog para mejorar su posicionamiento en los buscadores.

¿Cómo hacer una web optimizada para SEO?

Para hacer una web optimizada para SEO y que los usuarios visiten esta página debemos de realizar los siguientes pasos para conseguir tener un mejor posicionamiento web

Mejorar la accesibilidad a tu web

Esto lo conseguimos permitiendo los motores de búsqueda de contenido

Consigue una indexabilidad correcta

Se debe de seguir un orden para que resulte más fácil y que se pueda acceder más a esta, se ordena: según su contenido, autoridad y relevancia

Crea contenido de relevancia

Desde hace unos años para acá el contenido ha venido tomando relevancia, por eso si el contenido no es relevante va a ser muy difícil que aparezca de primero. La única forma es dando un cambio al contenido que se quiera publicar

Usa correctamente las meta etiquetas

Las meta etiquetas se utilizan para transmitir información a los buscadores sobre que trata una página, a la hora de ordenarla y mostrar los resultados.

Meta Etiquetas (HTML )

Las meta etiquetas (también llamadas: meta tags) se utilizan al crear páginas en HTML o XHTML. Estas etiquetas pueden describir el contenido de un sitio web con precisión. Este elemento te permite transmitir metadatos estructurados a través de la página web.

Principales Etiquetas Meta

Meta Title

La etiqueta title es la encargada de decirle a los motores de búsqueda la idea o tema principal sobre el que trata el contenido de la página que va a rastrear.

Meta Description

Complemento a la etiqueta Title, con la meta description le daremos una información algo más extensa a los motores de búsqueda del contenido de nuestra página.

Etiquetas Hs: Jerarquía de encabezados

Los encabezados servirán para jerarquizar y ordenar el contenido de una página, por orden de importancia, estos van desde el 1 hasta el 6 (h1, h2, h3…).

Etiqueta canonical

La meta etiqueta canonical es utilizada para decir a Google cual es la página original en el caso de encontrarnos con páginas similares o idénticas.

Etiqueta ALT

Esta es a la hora de realizar el posicionamiento de la imagen. Por esto haremos uso de la etiqueta ALT (texto alternativo), la cual ayudará a Google a saber de qué trata nuestra imagen.

Meta Robots

En esta le indicaremos a Google que contenido deber ser indexado y cual no, así como si los enlaces deben ser seguidos o no.

Open Graph

Las etiquetas Open Graph, son un protocolo que nos permite indicar a las redes sociales que información mostrar cada vez que se comparta un enlace de nuestro sitio web.

Relación de Meta Etiquetas y SEO

La relación entre estos dos es sumamente importante, ya que son elementos relevantes para la estrategia de SEO, una buena implementación de meta etiquetas ayuda a mejorar el posicionamiento en los resultados de búsqueda, lo cual es fundamental para aumentar el flujo de visitantes a un sitio web.
